List projects

RegVelo Gene-regulatory-informed dynamics of single cells

RegVelo 是一个 end-to-end 框架，将 splicing dynamics 与 gene regulatory priors 结合，用于推断单细胞 regulatory dynamics、latent time 和 perturbation effects。

software method single-cell grn
Neural crest regulatory circuits Single-cell multi-omics, spatial transcriptomics and systematic perturbation

该项目整合 multiome、Smart-seq3、MERSCOPE、Perturb-seq 与调控网络分析，用于解析 cranial neural crest migration 和 fate plasticity 的调控逻辑。

publication resource development multiomics
TEMPT Whole-genome EV-DNA methylation profiling

TEMPT 是面向 EV-DNA 的低输入、非亚硫酸氢盐全基因组甲基化测序方法，用于胃癌液体活检和细胞间通信特征识别。

method epigenomics liquid-biopsy publication
FINE-EM-seq Rapid isothermal methylome profiling for early embryos

FINE-EM-seq 是面向斑马鱼早期胚胎的快速等温扩增甲基组方法，配套提供 reproducibility analysis repo。

method epigenomics zebrafish publication
CIDER Interpretable meta-clustering for scRNA-seq integration and evaluation

CIDER 是一个用于单细胞整合数据 meta-clustering 与 integration evaluation 的可解释框架，已提供文档站点与可安装 R 包。

software single-cell integration resource
